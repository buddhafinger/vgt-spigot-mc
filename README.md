
Spigot Minecraft in Vagrant instance.

To use:
- Install Virtualbox / Vagrant.
- Clone this repo
- Enter dir and do vagrant up.
- Play.

Notes: THIS IS WORK IN PROGRESS / EARLY STAGES.
